# .vim
.vim stuff, better organized


## installation
1) clone this repo
2) create a sym link in the original .vimrc:

`ln -s <path_to_project>/.vimrc ~/.vimrc`
3) `. ~/.vimrc`


to add new vim-plugin:

1) Add Plug '<githubsn>/<githubRepo>' to your vimrc file.
2) Reload your vimrc or restart
3) Run :PlugInstall
